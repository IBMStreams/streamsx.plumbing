Namespace com.ibm.streamsx.plumbing.switches

com.ibm.streamsx.plumbing > com.ibm.streamsx.plumbing 0.5.1 > com.ibm.streamsx.plumbing.switches

Switch operators. Operators that provide switches for streams including a Changeover that submits tuples to one of two output ports depending if the switch is on or off.

ControlledSwitch and ControlledChangeover are switches whose state can be changed through control variables. This allows disconnected operators within the same application to control the state of switches. For example downstream processing can change an upstream flow through a controlled switch.

TraceLevelSwitch is a switch whose state can be changed via the application trace level of the PE it is a part of.

Operators

  • Changeover: Changeover switch.
  • ControlledChangeover: Controlled changeover switch.
  • ControlledSwitch: Controlled switch.
  • TraceLevelSwitch: Non-blocking switch set by application trace level.

Functions

  • getSwitchState(rstring): Get the state of a switch with control variable name.
  • registerSwitch(rstring, boolean): Register that an operator invocation will call setSwitchState(rstring, boolean) or getSwitchState(rstring).
  • setSwitchState(rstring, boolean): Set the state of switches with control variable name.
