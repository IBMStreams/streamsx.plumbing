Leadership election functionality.

Leadership election allows one of a number of operators to be nominated as the leader. Upon failure of the leader a new leader is nominated automatically from the remaining operators.

Leadership election can be used with redundant flows to ensure that only a single flow is producing output, such as sending SMS alerts.

Operators

  • LeadershipChangeover: Changeover switch that changes state based upon if the PE is marked as the leader of group.
  • LeadershipElection: Performs a leadership election between other operator invocations with the same group.
  • LeadershipSwitchWithReplay: Switch operator that only generates output when its PE is the leader of the group.

Types

  • LeaderStatus: Type of tuples submitted by LeadershipElection on its output port.
